라라벨을 백엔드 서버로 이용하고 플러터로 UI를 만들어 직접 플레이스토어에 배포해본 로또 번호 추천앱 입니다. 백엔드 서버는 AWS에 호스팅 하였으며 DB에 저장된 정보를 api를 통해 앱에 정보를 제공하고 있습니다. 만들면서 구글 애드몹을 이용한 광고도 붙여 보았고 로또 가 매주 토요일 추첨을 하게되면 앱에서도 새 당첨정보를 제공하여야 하는데 이부분은 크론탭을 이용하여 매주 토요일 오후 9:05분에 새 당첨정보를 가져와 DB에 저장하게 하여 당첨정보를 계속적으로 업데이트 하게 구현 하였습니다.

플러터로 구현한 폴라로이드 사진 앱 입니다. Provider 패턴을 공부하면서 만들게 되었고, 내 사진첩에 있는사진을 불러와 위젯만 보면 폴라로이드 사진을 찍은것같은 화면을 구현하였습니다. 또한 위젯 캡쳐를 사용해 캡쳐된 모습이 폴라로이드 사진을 인화한 것 같은 모습이 되게 하였으며, 갤러리에 캡쳐이미지를 저장할 수 있게 구현하였고 구글 폰트를 사용해 여러 글씨체도 제공하고 배경색상도 변경 할 수 있는 폴라로이드 어플을 구현하였습니다.
